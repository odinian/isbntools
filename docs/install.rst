

Install
=======

From the command line enter (in some cases you have to precede the
command with ``sudo``):


.. code-block:: bash

    $ pip install isbntools

or:

.. code-block:: bash

    $ easy_install isbntools

or:

.. code-block:: bash

    $ pip install isbntools-4.3.14.tar.gz

(first you have to download the file!)

You should check if the install was successful, by entering:

.. code-block:: bash

    $ isbntools
